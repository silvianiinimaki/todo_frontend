Sovelluksessa pystyy lisäämään tehtäviä listaan. Tehtäville voi määrittää deadlinen klikkaamalla kalenterista päivää, ja tehtävän tärkeyden voi määrittää klikkaamalla tähtiä.
Tehtävistä näkyy nyt aluksi vain otsikko ja checkbox, josta voi merkata tehtävän tehdyksi klikkaamalla.
Jos hiiren vie tehtävän päälle, ilmestyy laatikko, jossa listataan loput tiedot tehtävästä.
Poista valmiit -painikkeesta voi poistaa kaikki valmiiksi merkityt tehtävät.
